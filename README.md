# Motion craft

The custom LMS for motion design courses with admin panel.


## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform


## Tech Stack

- Laravel
- Livewire
- Tailwind
- Alpine
- Filament
- MySQL
